**AdGuard Browser Extension** is a high-performance browser extension designed to remove unwanted advertisements, block tracking technologies, and create a cleaner and more private web experience.

Using advanced filtering mechanisms, the extension detects and blocks different categories of ads, intrusive scripts, tracking requests, and unnecessary website components. AdGuard not only hides visible advertising but also helps prevent online services from collecting behavioral data that may be used to build detailed user profiles.

Compared with the complete AdGuard applications for Windows and macOS, the browser extension is completely free and available as an open source project. It delivers essential ad blocking and privacy protection capabilities directly within supported browsers.

> AdGuard Browser Extension follows a privacy-first approach: it does not collect personal information, record browsing activity, or participate in acceptable ads programs. The project continues to focus on secure browsing and user-controlled protection.

## <a name="installation"></a> Installation

### <a name="installation-chrome"></a> Chrome and Chromium-based browsers

Download and install AdGuard Browser Extension from the official [Chrome Web Store](https://agrd.io/extension_chrome).

### <a name="installation-firefox"></a> Firefox

Get the latest version of AdGuard Browser Extension through the official [Mozilla Add-ons website](https://agrd.io/extension_firefox).

### <a name="installation-opera"></a> Opera

The extension is distributed through the official [Opera Add-ons store](https://agrd.io/extension_opera).

### <a name="installation-edge"></a> Microsoft Edge

Install AdGuard Browser Extension directly from the official [Microsoft Store](https://agrd.io/extension_edge).
